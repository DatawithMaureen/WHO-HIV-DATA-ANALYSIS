
## WHO HIV DATA ANALYSIS DASHBOARD

## Project Overview
This project analyzes global HIV trends using WHO datasets to uncover insights into infection rates, mortality, treatment coverage, and survival outcomes globally.

The dashboard was developed using Power BI, DAX, and Power Query to support healthcare decision-making through data-driven insights.

## Key Insights
- HIV infections declined globally over time.
- African countries recorded the highest number of HIV cases and deaths.
- Increased ART treatment coverage contributed to reduced mortality rates.
- Some countries still experience high infection rates despite global progress.

## Tools Used
- Power BI
- DAX
- Power Query
- Excel

## Dashboard Features
- Executive Overview Dashboard
- Treatment & Survival Insights
- Recommendations & Conclusions
- Interactive slicers and filters
